# Playbook — Fases de Construção de Negócio Digital

Consulte este playbook durante cada heartbeat para identificar o que está faltando.
Compare o estado atual do negócio (business.md) contra cada fase e gere hipóteses
para os gaps mais críticos.

## Fase 0: Investigação de Mercado
- [ ] Tamanho do mercado endereçável (TAM/SAM/SOM) estimado com fontes
- [ ] Análise de concorrentes diretos e indiretos (preço, proposta, canais, pontos fracos)
- [ ] Validação de demanda (volume de busca, comunidades, reclamações existentes)
- [ ] Mapeamento de canais onde o ICP já está (grupos, fórums, redes sociais)
- [ ] Pesquisa de regulamentação aplicável (LGPD, CDC, setorial)

Skills aplicáveis: `/competitor-alternatives`, `/marketing-ideas`, `/content-strategy`

## Fase 1: Fundação da Marca
- [ ] Nome validado (disponibilidade de domínio, redes sociais, registro de marca)
- [ ] Domínio registrado
- [ ] Identidade visual mínima (cores, logo, tipografia)
- [ ] Contas em redes sociais criadas (Instagram, Twitter/X, LinkedIn, TikTok — conforme ICP)
- [ ] Email profissional configurado (domínio próprio)
- [ ] Página "sobre" ou bio consistente em todos os canais

Skills aplicáveis: `/copywriting`, `/copy-editing`

## Fase 2: Presença Digital Mínima
- [ ] Landing page publicada e acessível
- [ ] SEO básico (title, meta description, OG tags, sitemap)
- [ ] Analytics instalado (GA4, Clarity, ou equivalente)
- [ ] Pixel de conversão instalado (Meta, Google — conforme canal de aquisição)
- [ ] Formulário de captura funcional
- [ ] Lead magnet disponível
- [ ] Termos de uso e política de privacidade publicados

Skills aplicáveis: `/page-cro`, `/seo-audit`, `/ai-seo`, `/analytics-tracking`, `/schema-markup`, `/site-architecture`, `/lead-magnets`, `/signup-flow-cro`

## Fase 3: Aquisição e Growth
- [ ] Canal primário de aquisição ativo (ads, orgânico, parcerias)
- [ ] Conteúdo orgânico em redes sociais (calendário editorial mínimo)
- [ ] Estratégia de SEO/conteúdo (palavras-chave, blog posts)
- [ ] Presença em comunidades relevantes (Reddit, grupos Facebook, fórums)
- [ ] Programa de referral ou incentivo boca-a-boca
- [ ] Email marketing / nurturing configurado
- [ ] Parcerias estratégicas identificadas

Skills aplicáveis: `/paid-ads`, `/ad-creative`, `/social-content`, `/email-sequence`, `/cold-email`, `/launch-strategy`, `/referral-program`, `/programmatic-seo`, `/content-strategy`, `/free-tool-strategy`, `/marketing-psychology`

## Fase 4: Operação e Retenção
- [ ] Fluxo de entrega do serviço documentado e testável
- [ ] Sistema de cobrança/pagamento configurado
- [ ] Suporte ao cliente (canal e SLA definidos)
- [ ] Métricas de satisfação (NPS, CSAT)
- [ ] Loop de feedback do cliente → melhoria do produto
- [ ] Automações de retenção (emails, notificações)

Skills aplicáveis: `/churn-prevention`, `/onboarding-cro`, `/form-cro`, `/popup-cro`, `/paywall-upgrade-cro`, `/ab-test-setup`

## Fase 5: Escala
- [ ] Unit economics validados com dados reais
- [ ] Canais secundários de aquisição testados
- [ ] Automação de processos manuais
- [ ] Documentação operacional para delegação

Skills aplicáveis: `/pricing-strategy`, `/revops`, `/sales-enablement`, `/competitor-alternatives`
